# CodeInterviewAssist
[![en](https://img.shields.io/badge/lang-en-red.svg)](README.md)
[![ru](https://img.shields.io/badge/lang-ru-blue.svg)](README.ru.md)

Это форк [j4wg/interview-coder-withoupaywall-opensource](https://github.com/j4wg/interview-coder-withoupaywall-opensource) со следующими улучшениями:

- **Расширенная поддержка провайдеров**: Упрощённая архитектура провайдеров. Добавлены интеграции Ollama и OpenRouter.
- **Поддержка нескольких мониторов**: Полная поддержка 2K мониторов и систем с несколькими дисплеями (захватывает primary дисплей).
- **Предобработка скриншотов**: Добавлен pipeline предобработки скриншотов для сохранения функциональности на тёмных фонах. Возможности оптимизации для экономии токенов и скорости — кадрирование, грейскейл, компрессия.
- **Рефакторинг кода**: Перестроен ProcessingHelper на отдельные модули: промпты, обработка изображений, форматирование ответов.
- **Одна модель для всего**: В ветке `one-model-for-all` объединено распознавание и обработку в одной модели. `qwen3.5:9b` показывает отличные результаты.

### Скрытие окна терминала

Для скрытия окна терминала используйте [RBTray](https://github.com/benbuck/rbtray):

1. Запустите RBTray
2. Переведите фокус на нужное окно
3. Нажмите `Ctrl + Alt + Down Arrow` для сворачивания в системный трей

## Команды

Приложение использует неидентифицируемые глобальные горячие клавиши, которые не будут обнаружены браузерами или другими приложениями:

- Переключение видимости окна:: [Control or Cmd + B]
- Перемещение окна: [Control or Cmd + Arrow keys]
- Сделать скриншот: [Control or Cmd + H]
- Удалить последний скриншот: [Control or Cmd + L]
- Обработать скриншоты: [Control or Cmd + Enter]
- Начать новую задачу: [Control or Cmd + R]
- Выйти: [Control or Cmd + Q]
- Уменьшить прозрачность: [Control or Cmd + []
- Увеличить прозрачность: [Control or Cmd + ]]
- Уменьшить масштаб: [Control or Cmd + -]
- Сбросить масштаб: [Control or Cmd + 0]
- Увеличить масштаб: [Control or Cmd + =]

## Невидимость

Приложение невидимо для:

- Zoom версий ниже 6.1.6 (включительно)
- Всех браузерных программ для записи экрана
- Всех версий Discord
- Функции скриншотов Mac OS (Command + Shift + 3/4)
Примечание: Приложение НЕ невидимо для:
- Zoom версий 6.1.6 и выше https://zoom.en.uptodown.com/mac/versions (ссылка для отката Zoom при необходимости)
- Нативной записи экрана Mac OS (Command + Shift + 5)

## Запуск приложения

### Быстрый старт

1. Клонируйте репозиторий:

```bash
git clone https://github.com/Lanctole/interview-helper.git
cd interview-helper
```

2. Установите зависимости:

```bash
npm install
```

3. **Рекомендуется**: Очистите предыдущие сборки:

```bash
npm run clean
```

4. Запустите соответствующий скрипт для вашей платформы:

**For Windows:**
```bash
stealth-run.bat
```

**For macOS/Linux:**
```bash
# Make the script executable first
chmod +x stealth-run.sh
./stealth-run.sh
```

**ВАЖНО**: Окно приложения по умолчанию будет невидимым! Используйте Ctrl+B (или Cmd+B на Mac) для переключения видимости.

### Сборка дистрибутивов

Для создания установочных пакетов для распространения:

**For macOS (DMG):**
```bash
# Using npm
npm run package-mac

# Or using yarn
yarn package-mac
```

**For Windows (Installer):**
```bash
# Using npm
npm run package-win

# Or using yarn
yarn package-win
```
## Как это работает

1. **Начальная настройка**
   - Запустите невидимое окно
   - Выберете свой провайдер в настройках, вставьте ключ
   - Выберите предпочитаемую модель для извлечения, генерации решений и отладки

2. **Захват задачи**
   - Используйте сочетание [Control or Cmd + H] для создания скриншотов задач
   - Скриншоты автоматически добавляются в очередь (максимум 2)
   - При необходимости удалите последний скриншот с помощью [Control or Cmd + L]

3. **Обработка**
   - Нажмите  [Control or Cmd + Enter]  для анализа скриншотов

4. **Решение**
   - Просматривайте сгенерированные решения

5. **Управление окном**
   - Перемещайте окно с помощью [Control или Cmd + Стрелки]
   - Переключайте видимость с помощью [Control или Cmd + B]
   - Регулируйте прозрачность с помощью [Control или Cmd + [] и [Control или Cmd + ]]
   - Окно остаётся невидимым для указанных приложений для демонстрации экрана
   - Начинайте новую задачу с помощью [Control или Cmd + R]
